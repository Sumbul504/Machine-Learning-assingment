#  Machine Learning  Projects

##  Overview
This repository contains multiple machine learning projects based on real-world business problems. Each project uses a different model to explore how machine learning can be applied in practical scenarios.

---

##  Projects

###  Customer Churn Prediction
- **Model:** Random Forest  
- Predicts whether a customer is likely to leave  

###  Sales Forecasting
- **Model:** Gradient Boosting Regressor  
- Predicts future sales trends   

###  Credit Risk Prediction
- **Model:** Logistic Regression  
- Classifies customers into risk categories  

###  HR Attrition Prediction
- **Model:** LightGBM  
- Predicts employee turnover  

---

##  Workflow
- Data Cleaning  
- EDA & Visualization  
- Feature Engineering  
- Model Training & Evaluation  

---

##  Tools Used
Python, Pandas, NumPy, Matplotlib, Scikit-learn, LightGBM  

---

##  Key Takeaways
- Feature engineering plays a major role in performance  
- Different problems require different models  
- Machine learning can provide useful business insights  

---

##  Conclusion
These projects show how machine learning can be used to analyze data and make predictions that support better decision-making.
